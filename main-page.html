Cardbuster is a merge between action-adventure and deckbuilder genres. While keeping the strategic style of a deckbuilder, we wanted to make an action filled real-time game. In Cardbuster you use Cards which grant you abilities for a limited time to change the gameplay.

After implementing basic movement I have developed the architecture of the ability system. I have used Scriptable Objects as a way to store information about the abilities' both aesthetic and mechanical elements. Mechanical elements include damage, damageType, effectRadius, radiusAngle, abilityTarget, abilityDuration, attackSpeed etc. Aesthetic elements include name, abilityImage, vfx, sfx etc. Using a card utilizes this stored information to enable using the corresponding ability to either defeat enemies or overcome platforming challenges.

I also have made VFX such as ability effects. I have prepared sprites to use in Unity's Particle Effects and Visual Effect Graph tools working on Adobe Photoshop.

LIFE SOCKS PIC

Life Socks is 2021 Global Game Jam entry game for our team. Life Socks is a 2D-platformer-puzzle game in which you control a sock who has lost their pair trying to find them. Player has 2 item slots in which they can carry either keys to open doors or power-ups. There are power-ups such as 'Feet' which enable you to walk faster/jump and if you have 2 'Feet' items you can double-jump, 'Umbrella item enables you to glide long distances.

I took part in programming and level design of this game. I also did the implementation of the level in Unity.
